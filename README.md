# maxscript_calculator

3D 오브젝트의 길이를 측정해주는 툴입니다.

1. vertex 사이의 거리
2. edge 길이
3. polygon 넓이
4. volume
5. 중심 좌표

**2번, 3번의 경우에는 여러개를 선택했을 시에는 총 길이, 총 면적의 값을 함께 구해줍니다.
